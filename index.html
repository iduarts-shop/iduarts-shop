  • ID Techroom Guidelines

    The ID UArts Techroom is a student-run service that utilizes digital fabrication technology. We currently have a ULS VersaLaser 6.60 laser cutter, a Dimension SST 1200es 3D printer, and a ShopBot CNC router.

    We allow students and faculty to utilize these machines for prototyping and model-making purposes. Our submission process is as follows:

    • The customer reads the appropriate documentation on our site and prepares his/her file according to our standards.
    • The customer completes a Job Request Form
    • The customer will recieve an email if there are any errors in the file, requesting the customer fix and submit a new job request.
    • The file is run and the customer receives a confirmation e-mail with the final price and instructions on picking it up.
    • The customer goes to the 5th floor ID shop in the Terra building, pays for the file using credit card or Blackboard, and receives the finished piece(s).

    Please work in advance; pending queue status and project timing your file could take up to one week to cut.
